= 如何管理記錄與資料備份的內部管理SnapCenter
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
可在儲存系統和檔案系統層級、以及SAP HANA備份目錄中管理記錄和資料備份的管理。SnapCenter

主要或次要儲存設備上的Snapshot複本、以及SAP HANA目錄中對應的項目、會根據保留設定予以刪除。SAP HANA目錄項目也會在刪除備份和資源群組時刪除。
