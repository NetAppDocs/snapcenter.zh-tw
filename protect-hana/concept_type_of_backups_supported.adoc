= 支援的備份類型
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
備份類型指定您要建立的備份類型。支援SAP HANA資料庫的檔案型備份和Snapshot複本型備份類型。SnapCenter



== 檔案型備份

檔案型備份可驗證資料庫的完整性。您可以排定在特定時間間隔執行檔案型備份作業。只備份作用中的租戶。您無法從SnapCenter 無法還原及複製檔案型備份。



== Snapshot複製型備份

Snapshot複製型備份運用NetApp Snapshot複製技術、為SAP HANA資料庫所在的磁碟區建立線上唯讀複本。
