= 設定SnapCenter 以檢查可用的更新
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
本產品可定期與NetApp支援網站通訊、通知您有可用的軟體更新。SnapCenter您也可以建立排程、以指定接收可用更新資訊的時間間隔。

.步驟
. 在左導覽窗格中、選取 * 設定 * 。
. 在 *Settings* 頁面中，選擇 *Software*。
+
「可用軟體」頁面會顯示可用的外掛程式套件、可用版本及其安裝狀態。

. 選擇 *Check for updates* 以查看是否有更新版本的外掛程式套件可用。
. 選擇 *Schedule Updates* 以建立排程，指定您要接收可用更新相關資訊的時間間隔：
+
.. 在*檢查更新*中選取時間間隔。
.. 選取SnapCenter 「更新伺服器管理Windows」認證、然後按一下「*確定*」。
