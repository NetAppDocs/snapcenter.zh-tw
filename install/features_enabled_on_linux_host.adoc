= 安裝期間在 Linux 主機上啟用的功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter 伺服器會安裝在下列軟體套件下方、這些套件可能對疑難排解和主機系統維護有興趣。

* RabbitMQ
* 恩靈思
* Erlang
* 從 8.0.5 版開始的 .NET Framework ，包括所有後續的 .NET 8 修補程式
* Pam-devel
* PowerShell
