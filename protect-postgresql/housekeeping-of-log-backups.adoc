= SnapCenter 如何管理資料備份的管理工作
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter 負責管理儲存系統和檔案系統層級上的資料備份管理。

根據保留設定、主要或次要儲存設備上的快照及其在 PostgreSQL 目錄中對應的項目都會被刪除。
