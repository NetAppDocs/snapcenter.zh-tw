= 支援的備份類型
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
備份類型指定您要建立的備份類型。SnapCenter 支援 MySQL 資料庫的 Snapshot 複製型備份類型。



== Snapshot複製型備份

Snapshot 複製型備份利用 NetApp 快照技術、為 MySQL 資料庫所在的磁碟區建立線上唯讀複本。
